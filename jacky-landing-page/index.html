学友说，“这次演唱会取名‘经典'，是想做到自己的经典。年轻的时候，有很多原因没有达到自己认为的最完美，这次是要向自己的理想和梦想出发。”希望把过去一些留给自己印象深刻的经典表演，无论是歌曲、舞蹈或是一些场面集合整理后，在这次演唱会的舞台上重新呈现一次。早在1995年，学友哥就已经踏上澳洲这片土地开唱。2018年，歌神再次降临澳洲。张学友将会唱响自己的经典，而歌声也会成为我们的经典。

挑战最经典的“四面台”演唱会

本次巡演将挑战张学友自己内心中最经典的四面舞台。这是张学友继1995年第一场在香港红磡的四面台演唱会之后，首度再次开启四面台巡回演唱会，这是学友哥心目中的经典画面之一。

开办澳洲可容纳观众最多的演唱会

ACY稀万----张学友“A CLASSIC TOUR”2018世界巡演悉尼站，将打破尘封已久的澳洲演唱会历史！至今为止，张学友是唯一一位在澳洲开四面台演唱会的华人歌手，这也将是澳洲演唱会史上室内场馆的最大制作，单场可容纳观众数最多的演唱会！

打造澳洲室内场馆的最强制作的演唱会

为了呈现学友哥心中的精彩画面，这次巡演的舞台设计会以高制作规格，采用全机械控制。四面舞台是由多层次的LED屏幕构成，仔细看可以分为多个独立的LED立方体。舞台灯光也会伴随着LED的不同组合而变换。在理性和感性的不停交错中，走进时光隧道，与过往的自己相拥相逢，重回往事岁月！

携大型管弦乐团、专业舞者及特技演员的演唱会

本次巡演学友哥还将携手32人的大型乐团暨管弦乐团、30位专业舞者与特技演员登陆澳洲！吉他、架子鼓、管弦乐的客串仿佛让人在聆听一场高品质的演奏会。此外，学友哥在本次巡演的服装造型上也是颇费心思，与合作了二十年的美术总监一起为演唱会搭配出10套不同风格的服装，配合演唱会上不同的篇章出现。

张学友最新消息

「A CLASSIC TOUR 学友·经典」永恒的经典演唱会

集经典于一身、创经典的诞生，“金氏世界纪录神级人物”张学友，终结广大歌迷们5年的等待，张学友带着全新的「A CLASSIC TOUR 学友·经典世界巡回演唱会」，串联起全球歌迷的过去、现在与未来，共同写下全球华人音乐盛事，永恒的经典演唱会。

「A CLASSIC TOUR 学友·经典」北京首站完美演出

"歌神"张学友完成北京首站连3场的演出，他在舞台上感性的表示，「今天不是我的第三场演出，其实是第四场，第一场没有观众的彩排，我也是这样用力的表演」歌神更表示其实他已经这样练习演唱长达100多天，等同演出过百场了。学友也特别感谢所有的工作人员们，实现了他疯狂的梦想，把这个复杂的制作搭建出来，他表示，「谢谢大家帮我完成我的梦想，我的野心太大。」

「A CLASSIC TOUR 学友·经典世界巡回演唱会」北京首站登场

全球华人引颈期盼，“歌神”张学友的全新「A CLASSIC TOUR 学友·经典世界巡回演唱会」终于启航！睽违5年的全新巡演，首站于10月21日、22日、23日连续3晚在北京乐视体育生态中心盛大展开，挑战他心目中最经典的四面舞台，以全LED铺天盖地，打造360度超立体四面台，以32人的大型乐团暨管弦乐的编制，结合30位专业舞者与特技演员，演绎”歌神”张学友心中的40首经典，每一首歌随着情境的起伏，以世界级制作规格，呈现学友心目中的精彩画面，视觉与听觉上的完美结合，搭配10套服装造型，让上万名全球首场幸运观众为之惊呼，堪称「经典中的经典！

双喜临门「学友·经典台北站」6场全数秒杀售罄，并恭喜学友入围金马奖影帝

张学友「A CLASSIC TOUR 学友·经典世界巡回演唱会」将于明年2月11、12、13、17、18、19日，在台北小巨蛋开唱，今天(10/1)中午12点于拓元售票系统一次开卖6场，在3分钟内瞬间流量涌入超过20万人，爆冲人次破系统纪录，造成系统当机。开卖20分钟完售6场6万6千张票，系统紧急关闭处理，修复并确认所有结帐动作后，下午2:30重新开启售票页面，剩余票券于10分钟内全部完售！得知门票销售一空，有20万人同时上线抢票，创下流量的纪录，张学友表示，「帮我谢谢大家，我会尽力把最好的带上舞台，希望不会辜负大家的支持与期望。 」今天同时宣布张学友以「暗色天堂」入围第53届金马奖最佳男主角，学友亦表示：「非常荣幸可以入围金马奖，自己的努力被看到很开心，谢谢金马奖评委。」
